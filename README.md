# AICostGuard

See every AI subscription. Control spend. Reduce risk.

## Monorepo
- `web`: React 19 + Vite client
- `api`: Hono API server
- `packages/shared`: shared zod schemas, types, and envelope helpers

## Quick Start
1. `pnpm i`
2. `pnpm dev`

Web: `http://localhost:5173`
API: `http://localhost:4000`

## Verification
- `pnpm -r typecheck`
- `pnpm -r lint`
- `pnpm test`
- `pnpm build`
